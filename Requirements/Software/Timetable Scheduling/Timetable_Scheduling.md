**BT - L0 Timetable Scheduling**
  - **BC - L1.1 Create slot (Essential)**  
    The admin shall be able to create a slot by specifying its course and subject, date/day, start time, end time (or duration), and location.
  - **BAS - L1.2 Automatic scheduling (Desirable)**  
    The application shall generate staff and time-slot suggestions based on defined constraints and availability.
  - **BS - L1.3 Assign staff to slots (Essential)**  
    The admin will vet and, if there are any conflicts, the admin is able to manually overwrite the staff members to an available slot.
  - **BAT - L1.4 Approve Timetable (Desirable)**  
    The admin shall be able to approve and publish the final timetable to the respective staff.
