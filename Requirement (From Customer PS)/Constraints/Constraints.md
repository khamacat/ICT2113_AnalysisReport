# Constraints

## TS - L0 Software
The software shall support secure management of academic administration activities for staff and students.

### TAS - L1 Application Setup
The application can be run on Android and iOS devices.

### TAI - L1 App Icon
A web-based application requires an icon.

### TDS - L1 Data Storage
Web-based application data, such as academic, staff, and student information, shall be stored on a server within university grounds.

### TUA - L1 URL Address
The web-based application shall have a specific address/URL within the server.

### TUI - L1 User Interfaces
The system shall provide user interfaces for administrative staff and teaching staff to perform their academic tasks.

### TAC - L1 Access Control
The system shall control access to functions based on user roles.
